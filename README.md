# Syndication feed

A Kotlin library for reading RSS 2.0 and ATOM 1.0 syndication feeds.

## Syndication specifications

- [Atom 1.0](https://www.ietf.org/rfc/rfc4287)
- [RSS 2.0](http://www.rssboard.org/rss-specification)
